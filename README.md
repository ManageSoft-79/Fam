# Fam
Transactions overview tool

for DIY and common investors in India

**Load Mutual fund transactions from files**

Required columns in the files: Date, Folio Number, Name of the Fund, Order, Units, NAV

Supported files:
- CAMS and KFintech transaction statements
  - https://www.camsonline.com/Investors/Statements/Transaction-Details-Statement
  - https://mfs.kfintech.com/investor/General/InvestorTransactionReport
- Kuvera's Transaction csv export

Download links are also provided in the app in 'Download Links' section.

Requires full history. Load all files of same account together in Fam.

## Features
- Transactions overview
- Mutual funds overview
- Categories overview
- Fund wise transactions

**Planned features**:

- More transaction filetypes
- Category charts
